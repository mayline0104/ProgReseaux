Auteurs : LE POLLES Mayline - LOUKHNATI Mohamed Khalil
Projet : Programmation réseaux TP3 et TP4
Groupe : 1
Hexanome : H4111
Année : 2022-2023

Pour lancer le programme, entrez la commande "make".
Pour lancer le serveur, entrez la commande "./server2".
Pour lancer un client, entrez la commande "./client2 [adresse du serveur] [nom du client]".

Fonctionnalités implémentées :
    - Afficher tous les groupes : /all
    - Afficher tous les groupes auquel le client appartient : /grpin
    - Afficher tous les clients connectés : /users
    - Afficher l'historique du client : /history
    - Effacer l'historique du client : /clear
    - Envoyer un message à tous les clients : [message]
    - Envoyer un message privé : @[nom du destinataire] [message]
    - Créer un groupe : /create [nom du groupe]
    - Rejoindre un groupe : /join [nom du groupe]
    - Quitter un groupe : /leave [nom du groupe]
    - Quitter tous les groupes : /leave all
    - Envoyer un message à un groupe : /send [group name] [message]
    - Afficher la liste des commandes disponibles : /help
